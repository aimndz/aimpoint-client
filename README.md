# AimPoint

## Description
  Aimpoint is a blog website that allows users to read and comment on blog posts, while providing an admin side to create and manage content.
  
  [ğŸŒ Live Preview Here](https://aimpoint.vercel.app)
  
## Features
  ### Client Side:
  - User sign up and login
  - View blog posts
  - Create comments on posts
    
  ### Admin SIde:
  - Create, read, update, and delete blog posts
  - Publish and unpublish a post

## Technologies Used
  - TypeScript
  - React.js
  - React Router
  - Tailwind CSS
